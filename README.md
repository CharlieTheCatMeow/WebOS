# MeowOS!

A web based operating system! You can do stuff with it, using the apps listed below!

## Apps:
- Intro
- Notes
- Stopwatch
- To-Do List
- Terminal
- Calculator
- Music Player

## Features
- Clock
- Loading Screen
- Start Menu
- Interactive elements

Yes all very exciting I know.
